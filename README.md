<h3 align="left">Features:</h3>

- User management 
- Plant machines management
- Adding reports and searching in reports history 

<h3 align="left">Technologies:</h3>

- Java11
- Spring 5
- Spring Security 5
- Hibernate 5
- PostgreSQL 13
- JSTL
- Bootstrap 5
